# ATHLETICA AI - FITNESS TRACKER
Acest proiect este un tracker de fitness inteligent care foloseste AI pentru a oferi sfaturi personalizate.

## COMPONENTE
1. **Frontend**: Web interface (HTML/CSS/JS)
2. **Backend**: Spring Boot (Java)
3. **AI Service**: FastAPI (Python)
4. **Baza de Date**: MySQL (Clever Cloud)

## PORNIRE
- Ruleaza `start_all.bat` din radacina proiectului.
- Acceseaza `auth.html` in browser.
